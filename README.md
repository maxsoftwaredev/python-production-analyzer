# Python Production Analyzer

## Overview

This project simulates a simple industrial data analysis system for monitoring machine production performance.

It processes production logs from CSV files and generates analytical reports using Python and Pandas.

The goal is to demonstrate practical skills in:
- Data processing
- Python programming
- Modular architecture
- Basic reporting systems

---

## Features

- CSV data ingestion
- Production performance analysis
- Error tracking per machine
- Runtime aggregation
- Modular code structure
- Simple report generation

---

## Project Structure

data/ → input CSV files  
reports/ → output files  
src/ → application logic  
main.py → entry point  

---

## Technologies Used

- Python 3
- Pandas
- OOP principles
- File handling
